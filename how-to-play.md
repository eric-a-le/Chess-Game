# Run the Game
- Navigate to the folder containing the game files in the command line.
- If you have the virtual environment with pygame installed, activate the virtual environment.
- To run the game, type `python main.py`.

# Mouse Controls
Our game runs on drag and drop. In order to make a move, simply click and drag
a piece to the desired square. If the move is denied, then it is an illegal
move.
